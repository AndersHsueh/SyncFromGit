# SyncFromGit (기타저고)

Obsidian 볼트를 Git을 통해 동기화하여 여러 기기 간에 원활한 동기화를 가능하게 하는 강력한 Obsidian 플러그인입니다.

## 기능

### 1. Git 저장소 동기화
- 원격 Git 저장소 URL 구성
- 로컬 변경 사항을 원클릭으로 원격 저장소에 동기화
- 원격 저장소에서 최신 콘텐츠 가져오기

### 2. 파일 수준 동기화
- 파일 브라우저 컨텍스트 메뉴에 "Git에 커밋" 옵션 추가
- 개별 파일 또는 전체 폴더를 Git 저장소에 커밋
- 자동으로 `git add`, `git commit`, `git push` 실행

### 3. 저장소 관리
- 원격 Git 저장소를 로컬 볼트에 복제
- 브랜치 전환 (Obsidian 볼트 이름을 브랜치 이름으로 사용)
- 사이드바에서 동기화 작업

### 4. 다중 기기 지원
- Git 저장소를 원격 "클라우드 저장소"로 사용하여 기기 간 동기화
- 다중 브랜치 관리 (대화상자에서 선택)

## 기술 스택

- **TypeScript**: 타입 안전 개발 언어
- **Obsidian API**: Obsidian이 제공하는 플러그인 API
- **Git**: 버전 관리 및 동기화 메커니즘
- **Node.js**: 빌드 및 종속성 관리

## 설치 방법

1. 플러그인 파일을 Obsidian 볼트의 `.obsidian/plugins/` 디렉토리에 다운로드
2. Obsidian 재시작
3. 설정에서 플러그인 활성화

## 사용 방법

### Git 저장소 구성
1. 플러그인 설정 열기
2. Git 저장소 URL 입력
3. 설정 저장

### 파일 동기화
1. 파일 브라우저에서 파일 또는 폴더를 마우스 오른쪽 버튼으로 클릭
2. "Git에 커밋" 선택
3. 플러그인이 자동으로 원격 저장소에 변경 사항 푸시

### 원격 저장소 복제
1. 사이드바에서 "Git 저장소 가져오기" 버튼 클릭
2. Git 저장소 URL 입력
3. 체크아웃할 브랜치 선택
4. 플러그인이 저장소 내용을 로컬에 복제

### 수동 동기화
1. 사이드바의 동기화 버튼을 사용하여 수동으로 가져오기/푸시 작업

## 브랜치 전략

- 기본적으로 Obsidian 볼트 이름을 Git 브랜치 이름으로 사용
- 저장소에 여러 브랜치가 있는 경우 선택 대화상자 표시

## 보안

- 플러그인은 사용자가 액세스 권한을 가진 Git 저장소에서만 작동
- 사용자는 Git 저장소의 보안 및 액세스 권한을 확보할 책임이 있음
- 플러그인은 민감한 인증 정보를 수정하거나 전송하지 않음

## 참고 사항

- 시스템에 Git 명령줄 도구가 설치되어 있어야 함
- Git 저장소에 적절한 액세스 권한(SSH 키 또는 자격 증명) 필요
- 중요한 데이터의 정기적인 백업 권장
