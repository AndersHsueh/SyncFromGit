# SyncFromGit (吉他庫庫)

Obsidian のノート vaults を Git で同期し、複数のデバイス間でシームレスな同期を実現する強力な Obsidian プラグインです。

## 機能

### 1. Git リポジトリ同期
- リモートの Git リポジトリ URL を設定可能
- ローカルの変更を一クリックでリモートに同期
- リモートから最新のコンテンツを取得

### 2. ファイルレベル同期
- ファイルブラウザの右クリックメニューに「GIT にコミット」オプションを追加
- 選択した単一のファイルまたはディレクトリ全体を Git にコミット
- 自動的に `git add`、`git commit`、`git push` を実行

### 3. リポジトリ管理
- リモートの Git リポジトリをローカル vault にクローン
- ブランチ切り替え（Obsidian vault 名をブランチ名として使用）
- サイドバーからの同期操作

### 4. マルチデバイス対応
- Git リポジトリをリモート「クラウドストレージ」として使用し、デバイス間同期を実現
- マルチブランチ管理（ダイアログで選択可能）

## 技術スタック

- **TypeScript**: 型安全な開発言語
- **Obsidian API**: Obsidian が提供するプラグイン API
- **Git**: バージョン管理と同期メカニズム
- **Node.js**: ビルドと依存関係管理

## インストール方法

1. プラグインファイルを Obsidian vault の `.obsidian/plugins/` ディレクトリにダウンロード
2. Obsidian を再起動
3. 設定でプラグインを有効化

## 使い方

### Git リポジトリの設定
1. プラグイン設定を開く
2. Git リポジトリ URL を入力
3. 設定を保存

### ファイルの同期
1. ファイルブラウザでファイルまたはフォルダを右クリック
2. 「GIT にコミット」を選択
3. プラグインが自動的にリモートに変更をプッシュ

### リモートリポジトリのクローン
1. サイドバーの「GIT リポジトリを取得」ボタンをクリック
2. Git リポジトリ URL を入力
3. チェックアウトするブランチを選択
4. プラグインがリポジトリの内容をローカルにクローン

### 手動同期
1. サイドバーの同期ボタンを使用して手動でプル/プッシュ操作

## ブランチ戦略

- デフォルトで Obsidian vault 名を Git ブランチ名として使用
- リポジトリに複数のブランチがある場合、選択ダイアログを表示

## セキュリティ

- プラグインはユーザーがアクセス権限を持つ Git リポジトリのみ操作
- ユーザーは Git リポジトリのセキュリティとアクセス権限を確保する責任があります
- プラグインは機密の認証情報を変更または転送しません

## 注意事項

- システムに Git コマンドラインツールがインストールされている必要あり
- Git リポジトリには適切なアクセス権限（SSH キーまたは認証情報）が必要
- 重要なデータの定期的なバックアップを推奨
